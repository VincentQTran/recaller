# Recaller

Notion-Anki integration tool with semantic deduplication and AI-generated flashcards.

## Setup

```bash
conda activate recaller
pip install -e .
```

## Usage

```bash
recaller --help
recaller sync --dry-run
recaller status
```

## Configuration

Copy `.env.example` to `.env` and fill in your credentials:

```bash
cp .env.example .env
```
